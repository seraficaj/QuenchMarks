# QuenchMarks 

## About This Project

QuenchMarks is a web app that allows you to leave reviews of your favorite water bottles and track your daily hydration. 

## Technologies Used
- HTML/CSS/JavaScript
    - Jinja2 Templating Engine
- Python3
- SQLAlchemy
- Flask
